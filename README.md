# 🤖 ربات تبادل ممبر تلگرام

ربات تلگرامی برای مدیریت سیستم تبادل ممبر با سیستم سکه.

## ✨ امکانات

### کاربر عادی
- 🎁 عضویت در کانال‌ها و دریافت سکه
- 📦 سفارش ممبر با سکه
- 💸 انتقال سکه به کاربران دیگر
- 🎟️ استفاده از کد هدیه
- 👤 پروفایل با آمار کامل + تاریخچه تراکنش‌ها
- 👥 سیستم رفرال

### ادمین
- 📈 آمار کلی ربات
- 📦 مدیریت سفارش‌ها (لغو با دلیل + برگشت سکه)
- 📢 مدیریت کانال‌ها (افزودن / فعال‌سازی / غیرفعال‌سازی)
- 🎟️ مدیریت کدهای هدیه (ساخت / غیرفعال‌سازی)
- ⚙️ تنظیمات نرخ سکه‌ها
- 📣 پیام همگانی به تمام کاربران

## ⚙️ تنظیمات پیش‌فرض

| پارامتر | مقدار |
|---------|-------|
| سکه شروع | 50 |
| سکه هر عضویت | 1 |
| هزینه هر ممبر | 2 |
| سکه رفرال | 25 |

## 🚀 نصب و راه‌اندازی

### ۱. کلون پروژه
```bash
git clone https://github.com/yourusername/member-exchange-bot.git
cd member-exchange-bot
```

### ۲. نصب وابستگی‌ها
```bash
pip install -r requirements.txt
```

### ۳. تنظیم فایل .env
```bash
cp .env.example .env
```
فایل `.env` را باز کن و مقادیر را پر کن:
```
BOT_TOKEN=توکن_ربات_از_BotFather
ADMIN_IDS=آیدی_عددی_تلگرام_ادمین
```

> برای چند ادمین: `ADMIN_IDS=123456,789012`

### ۴. اجرا
```bash
python main.py
```

## ⚠️ نکات مهم

- ربات باید ادمین کانال‌هایی که اضافه میکنی باشه تا بتونه عضویت رو چک کنه
- آیدی عددی تلگرام خودت رو از [@userinfobot](https://t.me/userinfobot) بگیر
- فایل `bot.db` رو به `.gitignore` اضافه کن
